# Column Names and descriptions for King County Data Set

- **id** - unique identified for a house # yes
- **dateDate** - house was sold       # yes
- **pricePrice** - is prediction target    # yes
- **bedroomsNumber** - # of bedrooms      # yes
- **bathroomsNumber** - # of bathrooms    # yes
- **sqft_livingsquare** - footage of the home    # yes
- **sqft_lotsquare** - footage of the lot   # yes
- **floorsTotal** - floors (levels) in house  # yes
- **waterfront** - House which has a view to a waterfront  # no 
- **view** - quality of view         # no 
- **condition** - How good the condition is ( Overall )   # yes 
- **grade** - overall grade given to the housing unit, based on King County grading system  # yes
- **sqft_above** - square footage of house apart from basement  # no
- **sqft_basement** - square footage of the basement  # no
- **yr_built** - Built Year   # yes 
- **yr_renovated** - Year when house was renovated    # yes
- **zipcode** - zip    # yes
- **lat** - Latitude coordinate  # yes
- **long** - Longitude coordinate     # yes 
- **sqft_living15** - The square footage of interior housing living space for the nearest 15 neighbors     # no 
- **sqft_lot15** - The square footage of the land lots of the nearest 15 neighbors    # no
